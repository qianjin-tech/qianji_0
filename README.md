## 项目列表

### 1. Arduino Advanced Calculator ⭐
一个功能强大的 Arduino 科学计算器，支持科学计算、复数运算和矩阵运算。

**功能特性：**
- 基础四则运算（+、-、*、/、^）
- 科学函数（sin、cos、tan、log、ln、exp、sqrt、阶乘等）
- 复数运算（四则运算、模、辐角、三角函数）
- 矩阵运算（加减乘、转置、行列式、求逆、LU分解）
- 5x5 矩阵键盘 + LCD2004 显示
- 变量存储（A-J）

**硬件需求：**
- Arduino Uno/Nano
- LCD2004 (I2C)
- 5x5 矩阵键盘

📁 [进入项目](./Arduino_Calculator/)
